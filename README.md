# 귀화시험 종합평가 연습 앱

와이프가 맥북(또는 아이폰/아이패드)에 **앱처럼 설치**해서, **동기화 버튼**을 누르면
최신 문제를 받아 **모의고사·영역별 연습·작문/구술·오답노트·통계**를 볼 수 있는 앱입니다.
설치 파일 없이 브라우저에서 "설치" 한 번이면 독(Dock)에 앱 아이콘으로 들어갑니다. (PWA 방식)

---

## 1. 기능 한눈에

- **📝 모의고사** — 실제 시험처럼 객관식 36문항 + 60분 타이머 → 자동 채점, 합격선(60점) 표시
- **📚 영역별 연습** — 한국어·역사·정치/헌법·사회/문화·경제/지리, 한 문제씩 풀고 바로 해설
- **✍️ 작문·구술 연습** — 주제별 글쓰기(200자 글자수 표시) + 말하기 연습, 도움말 제공
- **🔁 오답 노트** — 틀린 문제 자동 저장, 다시 풀면 사라짐
- **📊 학습 통계** — 정답률, 영역별 성적, 모의고사 기록
- **오프라인 작동** — 한 번 동기화하면 인터넷 없이도 풀 수 있음

---

## 2. 와이프 맥북에 설치하기  (인터넷에 올린 뒤)

아래 3번에서 인터넷 주소(예: `https://내계정.github.io/gwiwha/`)가 만들어지면,
그 주소를 와이프에게 보내고 이렇게 설치합니다.

### 맥북 — Chrome / Edge
1. 그 주소를 크롬으로 엽니다.
2. 주소창 오른쪽 끝의 **설치 아이콘(⊕ 모니터 모양)** 클릭 → "설치".
3. 런치패드/독에 **"귀화시험"** 앱이 생깁니다.

### 맥북 — Safari (macOS Sonoma 이상)
1. 주소를 사파리로 엽니다.
2. 상단 메뉴 **파일 → Dock에 추가(Add to Dock)**.

### 아이폰 / 아이패드 — Safari
1. 주소를 사파리로 엽니다.
2. 아래 **공유 버튼 → 홈 화면에 추가**.

설치 후에는 앱을 열고 우측 상단 **⟳ 동기화**를 한 번 누르면 최신 문제를 받아옵니다.

---

## 3. 인터넷에 올리기 (호스팅) — 둘 중 하나

> 와이프 기기가 설치하고 동기화하려면 이 폴더가 인터넷 주소(HTTPS)로 떠 있어야 합니다.

### 방법 A) GitHub Pages — 추천 (주소 고정 + 업데이트 편함)
1. github.com 가입 → 새 저장소(repository) 만들기 (예: 이름 `gwiwha`, Public).
2. 이 폴더에서 아래를 실행 (이미 `git` 커밋은 준비돼 있음):
   ```
   git remote add origin https://github.com/<내아이디>/gwiwha.git
   git push -u origin main
   ```
3. 저장소 **Settings → Pages → Branch: main / (root)** 선택 후 저장.
4. 1~2분 뒤 `https://<내아이디>.github.io/gwiwha/` 주소가 생깁니다. → 이걸 와이프에게 전달.

**문제를 바꾼 뒤**에는 `git add . && git commit -m "문제 추가" && git push` 만 하면
와이프 앱에서 동기화 시(또는 다음 실행 시) 자동 반영됩니다.

### 방법 B) Netlify Drop — 가장 빠름 (오늘 바로)
1. https://app.netlify.com/drop 접속.
2. 이 폴더(`귀화시험앱`)를 통째로 브라우저에 **드래그&드롭**.
3. 즉시 `https://랜덤이름.netlify.app` 주소가 나옵니다. → 와이프에게 전달.
   (※ 무료 계정으로 "claim"하면 같은 주소로 계속 업데이트 가능)

## 4. 문제 추가 / 수정하기  ← 형님이 관리하는 부분

`questions.json` 파일 하나만 고치면 됩니다. 형식:

```json
{
  "id": "h08",                         // 겹치지 않는 고유 번호(아무거나)
  "category": "역사",                  // 공식 영역: 한국어 / 사회 / 문화 / 정치 / 경제 / 법 / 역사 / 지리 (작문 / 구술)
  "type": "mc",                        // 객관식
  "q": "문제 내용",
  "choices": ["보기1", "보기2", "보기3", "보기4"],
  "answer": 1,                         // 정답 보기 번호 (0=①, 1=②, 2=③, 3=④)
  "explanation": "해설(선택)"
}
```

- **작문 문제**: `"type": "writing"`, `q`(주제), `guide`(도움말). `choices`/`answer` 없음.
- **구술 문제**: `"type": "oral"`, `q`(질문), `guide`(도움말).
- 공식 견본문항(법무부 socinet.go.kr)이나 와이프가 가진 교재 문제를 여기에 추가하세요.
- 저장 후 호스팅에 반영(3번)하면 끝.

> ⚠️ 객관식만 자동 채점됩니다. 작문·구술은 정답이 없는 연습용(자기 점검)이라 점수에 안 들어갑니다.
> 실제 시험은 작문 10점 + 구술 25점이 포함된 100점 만점입니다(60점 이상 합격).

## 5. 파일 구조

| 파일 | 설명 |
|---|---|
| `index.html` | 앱 화면 구조 |
| `styles.css` | 디자인 |
| `app.js` | 동기화·퀴즈·채점·통계 로직 |
| `questions.json` | **문제은행 (여기만 고치면 됨)** |
| `sw.js` | 오프라인/자동 업데이트 |
| `manifest.webmanifest` | 앱 설치 정보 |
| `icon-*.png`, `icon.svg`, `apple-touch-icon.png` | 앱 아이콘 |
